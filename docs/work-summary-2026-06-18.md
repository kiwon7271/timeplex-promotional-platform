# Timeplex 플랫폼 작업 내역

**정리일:** 2026년 6월 18일  
**대상:** Timeplex Admin MVP (통합관리자 · 매장관리자 웹)

---

## 한 줄 요약

Timeplex 통합관리자와 매장관리자가 각자 역할에 맞게 **매장·입점·고객 대화·문의·약관** 등을 웹에서 관리할 수 있는 **기본 운영 화면(MVP)** 이 구축되었습니다.  
실제 메신저(WhatsApp 등) 연동이나 결제 기능은 아직 포함되지 않았으며, **목업(시범) 수준**으로 동작하는 기능도 있습니다.

---

## 공통 (로그인 · 보안 · 화면 기본)

| 항목 | 설명 |
|------|------|
| 로그인 | 이메일·비밀번호로 로그인 |
| 역할 자동 분기 | 통합관리자 → `/admin`, 매장관리자·직원 → `/store` |
| 접근 제한 | 로그인하지 않으면 관리 화면에 들어갈 수 없음 |
| 데이터 권한 | 통합관리자는 전체 데이터, 매장 사용자는 **본인 매장 데이터만** 조회·수정 |
| 반응형 화면 | PC·태블릿·모바일에서 레이아웃이 맞게 조정됨 |
| 공통 알림창 | 브라우저 기본 `alert` 대신 **알림·확인 팝업** 사용 |
| 아이콘 | 화면 전반에 통일된 아이콘 적용 |

## 통합관리자 (Timeplex 본사)

통합관리자는 플랫폼 전체를 조망·운영합니다.

### 대시보드
- 전체 매장 수, 대기 중인 입점 신청, 진행 중 대화, 이번 달 메시지 사용량을 한눈에 확인

### 매장 관리
- 등록된 매장 목록 조회
- 매장 상세 정보 확인 및 상태 관리

### 입점 관리
- 신규 입점 신청 목록 확인
- 승인·거절 등 입점 처리

### 매장 수요 · 방문자 통계
- 매장별 수요·방문 관련 통계 화면 (MVP 골격)

### 대화 로그
- 매장·고객 간 대화 내역 **조회 전용** (통합관리자는 메시지를 보내지 않음)
- 매장·대화방 선택 후 메시지 열람

### 채팅 사용량
- 매장별·월별 메시지 사용량 확인

### 매장 문의
- 매장에서 올린 문의 목록 확인 및 답변 처리

### 설정 — 동의/고지 약관 관리
- 매장관리자에게 보여줄 **동의·고지 문구** 등록·수정·삭제
- 제목, 내용, 버전, 활성/비활성 설정
- **버전을 올리면** 매장은 **다시 동의**해야 고객 대화를 이용할 수 있음

### 동의 증빙 (데이터)
- 매장이 약관에 동의한 기록(매장, 약관, 버전, 동의자, 동의 시각)이 **데이터베이스에 저장**됨
- 통합관리자는 시스템상 조회 권한 보유 (화면 목록 UI는 추후 추가 가능)

## 매장관리자 · 매장 직원

매장관리자와 직원은 **자기 매장** 업무만 처리합니다.

### 대시보드
- 채팅 서비스 운영 여부, 요금제, 직원 수, 진행 중 대화 수
- 제출 서류 상태 요약 (대기·승인·반려)

### 매장 정보
- 매장 기본 정보 조회·수정

### 직원 관리
- 매장 직원 목록 확인
- 요금제에 따른 직원 수 제한 반영

### 예약 링크
- 고객에게 보낼 예약 링크 등록·관리 (예: 네이버·카카오 등)

### 서류
- 매장 관련 서류 이미지 업로드 (jpg·png, 최대 5MB)
- 제출·승인 상태 확인

### 고객 대화 ⭐ (핵심 기능)

| 기능 | 설명 |
|------|------|
| 대화 목록 | 고객별 대화방 목록, 최근 메시지 순 정렬 |
| 검색·필터 | 이름·전화·이메일 검색, 채널(WhatsApp 등) 필터 |
| 메시지 보내기 | 텍스트, **이미지 첨부**, **예약 링크 첨부** |
| 예약 링크 말풍선 | 보낸 링크가 말풍선으로 표시되며, 클릭 시 새 탭에서 열림 |
| 스크롤 | 메시지 전송·대화방 진입 시 대화창이 **맨 아래**로 이동 |
| 한글 입력 | 한글 입력 중 엔터를 눌러도 **메시지가 두 번** 보내지지 않도록 처리 |

#### 약관 동의 (필수)
- **고객 대화 페이지에 들어가기 전**, 통합관리자가 설정한 **동의/고지 약관에 반드시 동의**해야 함
- 동의하지 않으면 **채팅을 시작할 수 없음**
- 동의 내용은 서버에 기록되어 증빙으로 남음
- 이미 동의한 뒤에는 페이지 헤더 우측 **「동의 약관 N」** 버튼으로 **약관 내용·동의 일시**를 다시 확인 가능

#### 공지 팝업
- 매장 영역 접속 시, 통합관리자가 등록한 **활성 공지** 내용을 팝업으로 표시
- 「확인」「오늘 하루 보지 않기」 선택 가능
- 고객 대화 페이지에서는 필수 동의 화면과 겹치지 않도록 공지 팝업은 표시하지 않음

### 문의
- Timeplex 통합관리자에게 문의 등록·목록 확인

## 아직 포함되지 않은 것 (MVP 제외)

- 실제 WhatsApp / LINE / Instagram 등 **외부 메신저 연동**
- **실시간** 채팅 (Supabase Realtime — 추후 연동 예정)
- 실제 **번역 API** 연동
- 결제, PDF·OCR, 푸시 알림, 최종 브랜드 디자인 등
